## <a name="example"></a><span data-ttu-id="cec1d-106">Esempio</span><span class="sxs-lookup"><span data-stu-id="cec1d-106">Example</span></span>  
 <span data-ttu-id="cec1d-107">L'esempio seguente genera l'errore CS0050 perché non è specificato alcun modificatore di accessibilità per `MyClass` e viene quindi usata l'impostazione predefinita per l'accessibilità `private`.</span><span class="sxs-lookup"><span data-stu-id="cec1d-107">The following sample generates CS0050 because no accessiblity modifier is supplied for `MyClass` and its accessibility therefore defaults to `private`.</span></span>

```csharp  
{  
    public static MyClass MyMethod()   // CS0050  
    {  
        return new MyClass();  
    }  
  
    public static void Main() { }  
}  
```
