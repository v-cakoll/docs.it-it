<span data-ttu-id="ae316-103">Firma per l'elemento inizializzatore non corretta nella corrispondenza migliore del metodo di overload per 'metodo'.</span><span class="sxs-lookup"><span data-stu-id="ae316-103">The best overloaded method match for 'method' has wrong signature for the initializer element.</span></span> <span data-ttu-id="ae316-104">Il metodo Add inizializzabile deve essere un metodo di istanza accessibile.</span><span class="sxs-lookup"><span data-stu-id="ae316-104">The initializable Add must be an accessible instance method.</span></span>
