# <a name="compiler-error-cs0507"></a><span data-ttu-id="cfa58-102">Errore del compilatore CS0507</span><span class="sxs-lookup"><span data-stu-id="cfa58-102">Compiler Error CS0507</span></span>

## <a name="example"></a><span data-ttu-id="cfa58-105">Esempio</span><span class="sxs-lookup"><span data-stu-id="cfa58-105">Example</span></span>  
 <span data-ttu-id="cfa58-106">L'esempio seguente genera l'errore CS0507.</span><span class="sxs-lookup"><span data-stu-id="cfa58-106">The following sample generates CS0507.</span></span>  
  
```csharp  
// CS0507.cs  
abstract public class clx  
{  
   virtual protected void f() {}  
}  
  
public class cly : clx  
{  
   public override void f() {}   // CS0507  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="cfa58-107">Esempio</span><span class="sxs-lookup"><span data-stu-id="cfa58-107">Example</span></span>  
 <span data-ttu-id="cfa58-108">L'errore CS0507 può verificarsi anche se una classe tenta di eseguire l'override di un metodo contrassegnato come `protected internal` definito nei metadati di riferimento.</span><span class="sxs-lookup"><span data-stu-id="cfa58-108">CS0507 can also occur if a class attempts to override a method marked as `protected internal` defined in referenced metadata.</span></span> <span data-ttu-id="cfa58-109">In questa situazione, il metodo sottoposto a override deve essere contrassegnato come `protected`.</span><span class="sxs-lookup"><span data-stu-id="cfa58-109">In this situation, the overriding method should be marked as `protected`.</span></span>

```csharp  
{  
   virtual protected internal void f() {}  
}  
```

 <span data-ttu-id="cfa58-111">L'esempio seguente genera l'errore CS0507.</span><span class="sxs-lookup"><span data-stu-id="cfa58-111">The following sample generates CS0507.</span></span>
