# <a name="compiler-error-cs0031"></a><span data-ttu-id="eedea-102">Errore del compilatore CS0031</span><span class="sxs-lookup"><span data-stu-id="eedea-102">Compiler Error CS0031</span></span>

<span data-ttu-id="eedea-103">Non è possibile convertire il valore costante 'value' in 'type'.</span><span class="sxs-lookup"><span data-stu-id="eedea-103">Constant value 'value' cannot be converted to a 'type'.</span></span>

<span data-ttu-id="eedea-104">Si è provato ad assegnare un valore a una variabile il cui tipo non può archiviare il valore.</span><span class="sxs-lookup"><span data-stu-id="eedea-104">An attempt was made to assign a value to a variable whose type cannot store the value.</span></span> <span data-ttu-id="eedea-105">Per altre informazioni, vedere [Tipi](../programming-guide/types/index.md).</span><span class="sxs-lookup"><span data-stu-id="eedea-105">For more information, see [Types](../programming-guide/types/index.md).</span></span>

<span data-ttu-id="eedea-106">L'esempio seguente genera l'errore CS0031 nei contesti selezionati e deselezionati:</span><span class="sxs-lookup"><span data-stu-id="eedea-106">The following sample generates CS0031 in both checked and unchecked contexts:</span></span>

```csharp
// CS0031.cs
namespace CS0031
{
    public class Program
    {
        public static void Main()
        {
            int num = (int)2147483648M; //CS0031
            // Try using a larger numeric type instead.
            // long num = (long)2147483648M; //CS0031

            const decimal d = -10M; // Decimal literal
            unchecked
            {
                const byte b = (byte)d; // CS0031
                // For small values try using a signed byte instead.
                // const sbyte b = (sbyte)d;
            }
        }
    }
}
```
