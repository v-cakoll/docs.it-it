# <a name="compiler-warning-level-1-cs1592"></a>Avviso del compilatore (livello 1) CS1592
Un codice XML con un formato non corretto è stato rilevato nel file dei commenti incluso -- 'reason'  
  
 Un problema, segnalato come ***motivo***, è stato trovato nel file specificato per il [ \<includono >](../../csharp/programming-guide/xmldoc/include.md) tag.
