# <a name="compiler-error-cs1107"></a><span data-ttu-id="ba887-102">Errore del compilatore CS1107</span><span class="sxs-lookup"><span data-stu-id="ba887-102">Compiler Error CS1107</span></span>
<span data-ttu-id="ba887-103">Un parametro può avere un solo modificatore 'modifier name'.</span><span class="sxs-lookup"><span data-stu-id="ba887-103">A parameter can only have one 'modifier name' modifier.</span></span>  
  
 <span data-ttu-id="ba887-104">È ad esempio un errore i modificatori di parametro `this`, `ref`, `in`, e `out` venga visualizzato più di una volta in una definizione di parametro.</span><span class="sxs-lookup"><span data-stu-id="ba887-104">It is an error for parameter modifiers such as `this`, `ref`, `in`, and `out` to appear more than one time in a parameter definition.</span></span>  
  
## <a name="example"></a><span data-ttu-id="ba887-105">Esempio</span><span class="sxs-lookup"><span data-stu-id="ba887-105">Example</span></span>  
 <span data-ttu-id="ba887-106">L'esempio seguente genera l'errore CS1107:</span><span class="sxs-lookup"><span data-stu-id="ba887-106">The following example generates CS1107:</span></span>  
  
```csharp  
// cs1107.cs  
public static class Test
{
    // Extension methods.  
    public static void TestMethod(this this t) { } // CS1107  

}

public class TestTwo
{ 
    // Regular Instance Method  
    public void TestMethod(ref ref int i) { } // CS1107  

    // Regular Instance Method  
    public void TestMethod(in in double d) { } // CS1107  
}
```
