# <a name="compiler-error-cs1040"></a>Errore del compilatore CS1040
Le direttive del preprocessore devono trovarsi il primo carattere non spazio vuoto su una riga  
  
 In una riga Ã¨ stata trovata una [direttiva per il preprocessore](../../csharp/language-reference/preprocessor-directives/index.md) che non rappresenta il primo token della riga. Una direttiva deve essere il primo token presente nella riga.  
  
 L'esempio seguente genera l'errore CS1040:  
  
```csharp  
// CS1040.cs  
/* Define a symbol, X */ #define X   // CS1040  
  
// try the following two lines instead  
// /* Define a symbol, X */  
// #define X  
  
public class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
