# <a name="compiler-error-cs0066"></a><span data-ttu-id="fbc75-102">Errore del compilatore CS0066</span><span class="sxs-lookup"><span data-stu-id="fbc75-102">Compiler Error CS0066</span></span>
<span data-ttu-id="fbc75-103">'event': l'evento deve essere di un tipo delegato</span><span class="sxs-lookup"><span data-stu-id="fbc75-103">'event': event must be of a delegate type</span></span>  
  
 <span data-ttu-id="fbc75-104">La parola chiave event richiede un tipo [delegate](../language-reference/builtin-types/reference-types.md) .</span><span class="sxs-lookup"><span data-stu-id="fbc75-104">The event keyword requires a [delegate](../language-reference/builtin-types/reference-types.md) type.</span></span> <span data-ttu-id="fbc75-105">Per ulteriori informazioni, vedere [eventi](../programming-guide/events/index.md) e [delegati](../programming-guide/delegates/index.md).</span><span class="sxs-lookup"><span data-stu-id="fbc75-105">For more information, see [Events](../programming-guide/events/index.md) and [Delegates](../programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="fbc75-106">L'esempio seguente genera l'errore CS0066:</span><span class="sxs-lookup"><span data-stu-id="fbc75-106">The following sample generates CS0066:</span></span>  
  
```csharp  
// CS0066.cs  
using System;  
  
public class EventHandler  
{  
}  
  
// to fix the error, remove the event declaration and the  
// EventHandler class declaration, and uncomment the following line  
// public delegate void EventHandler();  
  
public class a  
{  
   public event EventHandler Click;   // CS0066  
  
   private void TestMethod()  
   {  
      if (Click != null)  
         Click();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
