# <a name="compiler-error-cs0438"></a><span data-ttu-id="94267-102">Errore del compilatore CS0438</span><span class="sxs-lookup"><span data-stu-id="94267-102">Compiler Error CS0438</span></span>
<span data-ttu-id="94267-103">Il tipo 'type' in 'module_1' è in conflitto con lo spazio dei nomi 'namespace' in 'module_2'.</span><span class="sxs-lookup"><span data-stu-id="94267-103">The type 'type' in 'module_1' conflicts with the namespace 'namespace' in 'module_2'.</span></span>  
  
 <span data-ttu-id="94267-104">Questo errore si verifica quando un tipo in un file di origine è in conflitto con uno spazio dei nomi in un altro file di origine.</span><span class="sxs-lookup"><span data-stu-id="94267-104">This error occurs when a type in a source file conflicts with a namespace in another source file.</span></span> <span data-ttu-id="94267-105">Questo accade solitamente quando uno o entrambi gli elementi provengono da un modulo aggiunto.</span><span class="sxs-lookup"><span data-stu-id="94267-105">This typically happens when one or both come from an added module.</span></span> <span data-ttu-id="94267-106">Per risolvere il problema, rinominare il tipo o lo spazio dei nomi che ha causato il conflitto.</span><span class="sxs-lookup"><span data-stu-id="94267-106">To resolve, rename the type or the namespace that caused the conflict.</span></span>  
  
 <span data-ttu-id="94267-107">L'esempio seguente genera l'errore CS0438:</span><span class="sxs-lookup"><span data-stu-id="94267-107">The following example generates CS0438:</span></span>  
  
 <span data-ttu-id="94267-108">Compilare innanzitutto il file:</span><span class="sxs-lookup"><span data-stu-id="94267-108">Compile this file first:</span></span>  
  
```csharp  
// CS0438_1.cs  
// compile with: /target:module  
public class Util  
{  
   public class A { }  
}  
```  
  
 <span data-ttu-id="94267-109">Compilare quindi il file:</span><span class="sxs-lookup"><span data-stu-id="94267-109">Then compile this file:</span></span>  
  
```csharp  
// CS0438_2.cs  
// compile with: /target:module  
namespace Util   
{  
   public class A { }  
}  
```  
  
 <span data-ttu-id="94267-110">Infine compilare il file:</span><span class="sxs-lookup"><span data-stu-id="94267-110">And then compile this file:</span></span>  
  
```csharp

// compile with: /addmodule:CS0438_1.netmodule /addmodule:CS0438_2.netmodule  
using System;  
public class Test  
{  
   public static void Main() {  
      Console.WriteLine(typeof(Util.A));   // CS0438  
   }  
}  
```
