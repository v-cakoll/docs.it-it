## <a name="example"></a><span data-ttu-id="5002d-109">Esempio</span><span class="sxs-lookup"><span data-stu-id="5002d-109">Example</span></span>

```csharp  
    delegate void D<T>();  
    public static void GenericMethod<T>() { }

       // Try the following line instead

    }  
}  
```
