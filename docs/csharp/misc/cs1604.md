# <a name="compiler-error-cs1604"></a>Errore del compilatore CS1604
Non è possibile assegnare a 'variable' perché è di sola lettura  
  
 È stata effettuata un'assegnazione a una variabile di sola lettura. Per evitare questo errore, non assegnare o provare a modificare questa variabile in questo contesto.
