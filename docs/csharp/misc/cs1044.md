# <a name="compiler-error-cs1044"></a>Errore del compilatore CS1044
Impossibile utilizzare pi√π di un tipo nelle istruzioni for, using, fixed e nelle dichiarazioni  
  
 Il compilatore ha trovato un'istruzione non valida.  
  
 L'esempio seguente genera l'errore CS1044:  
  
```csharp  
// CS1044.cs  
using System;  
  
public class MyClass : IDisposable  
{  
   public void Dispose()  
   {  
      Console.WriteLine("Res1.Dispose()");  
   }  
  
   public static void Main()  
   {  
      using (MyClass mc1 = new MyClass(),  
             MyClass mc2 = new MyClass())   // CS1044, remove an instantiation  
      {  
      }  
   }  
}  
```
