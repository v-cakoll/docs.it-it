# <a name="compiler-error-cs1619"></a>Errore del compilatore CS1619
impossibile creare file temporaneo 'filename' -- reason  
  
 Il compilatore non è riuscito a creare un file temporaneo per il motivo specificato (ad esempio, il disco è pieno).
