# <a name="compiler-error-cs0669"></a><span data-ttu-id="249e9-102">Errore del compilatore CS0669</span><span class="sxs-lookup"><span data-stu-id="249e9-102">Compiler Error CS0669</span></span>

<span data-ttu-id="249e9-103">Una classe con l'attributo ComImport non può avere un costruttore definito dall'utente.</span><span class="sxs-lookup"><span data-stu-id="249e9-103">A class with the ComImport attribute cannot have a user-defined constructor.</span></span>

<span data-ttu-id="249e9-104">Il livello di interoperabilità COM in Common Language Runtime fornisce il costruttore per le classi [ComImport](xref:System.Runtime.InteropServices.ComImportAttribute) .</span><span class="sxs-lookup"><span data-stu-id="249e9-104">The COM interop layer in the common language runtime supplies the constructor for [ComImport](xref:System.Runtime.InteropServices.ComImportAttribute) classes.</span></span> <span data-ttu-id="249e9-105">Di conseguenza, è possibile usare un oggetto COM come oggetto gestito nel runtime.</span><span class="sxs-lookup"><span data-stu-id="249e9-105">Consequently, a COM object can be used as a managed object in the runtime.</span></span>

<span data-ttu-id="249e9-106">L'esempio seguente genera l'errore CS0669:</span><span class="sxs-lookup"><span data-stu-id="249e9-106">The following sample generates CS0669:</span></span>

```csharp
// CS0669.cs
using System.Runtime.InteropServices;
[ComImport, Guid("00000000-0000-0000-0000-000000000001")]
class TestClass
{
   TestClass()   // CS0669, delete constructor to resolve
   {
   }

   public static void Main()
   {
   }
}
```
