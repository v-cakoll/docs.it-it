## <a name="example"></a><span data-ttu-id="2fd22-107">Esempio</span><span class="sxs-lookup"><span data-stu-id="2fd22-107">Example</span></span>  
 <span data-ttu-id="2fd22-108">L'esempio seguente genera l'errore CS3019:</span><span class="sxs-lookup"><span data-stu-id="2fd22-108">The following sample generates CS3019:</span></span>  
  
```csharp  
// CS3019.cs  
// compile with: /W:2  
  
using System;  
  
[assembly: CLSCompliant(true)]  
  
// To fix the error, remove the next line

    [CLSCompliant(false)]  // CS3019

    {  
    }

    {  
    }  
}  
```
