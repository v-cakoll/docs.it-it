<span data-ttu-id="95126-103">Impossibile convertire il valore Null nel parametro di tipo 'name' perché potrebbe essere un tipo valore non nullable.</span><span class="sxs-lookup"><span data-stu-id="95126-103">Cannot convert null to type parameter 'name' because it could be a non-nullable value type.</span></span> <span data-ttu-id="95126-104">In alternativa, considerare l'uso di default('T').</span><span class="sxs-lookup"><span data-stu-id="95126-104">Consider using default('T') instead.</span></span>

## <a name="example"></a><span data-ttu-id="95126-108">Esempio</span><span class="sxs-lookup"><span data-stu-id="95126-108">Example</span></span>  
 <span data-ttu-id="95126-109">L'esempio seguente genera l'errore CS0403.</span><span class="sxs-lookup"><span data-stu-id="95126-109">The following sample generates CS0403.</span></span>  
  
```csharp  
// CS0403.cs  
// compile with: /target:library  
class C<T>  
{  
   public void f()  
   {  
      T t = null;  // CS0403  
      T t2 = default(T);   // OK  
    }  
}  
  
class D<T> where T : class   
{  
   public void f()  
   {  
      T t = null;  // OK  
    }  
}  
```
