 <span data-ttu-id="48c7f-108">Il codice seguente genera l'errore CS1931 perché l'identificatore `x` viene usato sia come variabile locale in `Main` , sia come variabile di intervallo nell'espressione di query:</span><span class="sxs-lookup"><span data-stu-id="48c7f-108">The following code generates CS1931 because the identifier `x` is used both as a local variable in `Main` and as the range variable in the query expression:</span></span>

    {  
        static void Main()

    }
