# <a name="compiler-error-cs0601"></a>Errore del compilatore CS0601
L'attributo DllImport deve essere specificato in un metodo contrassegnato come 'static' ed 'extern'  
  
 L'attributo `DllImport` Ã¨ stato usato in un metodo che non aveva le parole chiave di accesso corrette.  
  
 L'esempio seguente genera l'errore CS0601:  
  
```csharp  
// CS0601.cs  
using System.Runtime.InteropServices;  
using System.Text;  
  
public class C  
{  
   [DllImport("KERNEL32.DLL")]  
   extern int GetCurDirectory(int bufSize, StringBuilder buf);   // CS0601  
   // Try the following line instead:  
   // static extern int GetCurDirectory(int bufSize, StringBuilder buf);  
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
   }  
}  
```
