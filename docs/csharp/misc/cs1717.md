# <a name="compiler-warning-level-3-cs1717"></a><span data-ttu-id="5703b-102">Avviso del compilatore (livello 3) CS1717</span><span class="sxs-lookup"><span data-stu-id="5703b-102">Compiler Warning (level 3) CS1717</span></span>
<span data-ttu-id="5703b-103">Assegnazione fatta alla stessa variabile. Si intendeva assegnare qualcos'altro?</span><span class="sxs-lookup"><span data-stu-id="5703b-103">Assignment made to same variable; did you mean to assign something else?</span></span>

 <span data-ttu-id="5703b-105">L'avviso può essere generato da diversi errori comuni:</span><span class="sxs-lookup"><span data-stu-id="5703b-105">Several common mistakes can generate this warning:</span></span>

-   <span data-ttu-id="5703b-108">Errori di digitazione.</span><span class="sxs-lookup"><span data-stu-id="5703b-108">Mistyping.</span></span> <span data-ttu-id="5703b-109">È probabile che si volesse scrivere `a = b`.</span><span class="sxs-lookup"><span data-stu-id="5703b-109">You probably meant to say `a = b`.</span></span>
