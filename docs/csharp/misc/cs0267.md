# <a name="compiler-error-cs0267"></a>Errore del compilatore CS0267
Il modificatore partial può trovarsi solo immediatamente prima di 'class', 'struct' o 'interface'  
  
 Il posizionamento del modificatore **partial** non è corretto nella dichiarazione di classe, struct o interfaccia. Per risolvere l'errore, riordinare i modificatori. Per altre informazioni, vedere [Classi e metodi parziali](../programming-guide/classes-and-structs/partial-classes-and-methods.md).  
  
 L'esempio seguente genera l'errore CS0267:  
  
```csharp  
// CS0267.cs  
public partial class MyClass  
{  
   public MyClass()  
   {  
   }  
}  
  
partial public class MyClass  // CS0267  
// Try this line instead:  
// public partial class MyClass  
{  
   public void Foo()  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
