# <a name="compiler-error-cs1535"></a>Errore del compilatore CS1535
L'operatore unario di overload 'operator' accetta un parametro  
  
 La definizione di unario [operatore](../../csharp/language-reference/operators/operator-overloading.md) deve accettare un parametro.  
  
## <a name="example"></a>Esempio  
 L'esempio seguente genera l'errore CS1535:  
  
```csharp  
// CS1535.cs  
class MyClass  
{  
    // uncomment the method parameter to resolve CS1535  
    public static MyClass operator ++ (/*MyClass MC1*/)   // CS1535  
    {  
        return new MyClass();  
    }  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
