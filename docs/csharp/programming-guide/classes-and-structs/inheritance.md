 <span data-ttu-id="810b9-114">Quando si definisce una classe derivandola da un'altra classe, la classe derivata acquista implicitamente tutti i membri della classe di base, con l'eccezione dei costruttori e dei finalizzatori.</span><span class="sxs-lookup"><span data-stu-id="810b9-114">When you define a class to derive from another class, the derived class implicitly gains all the members of the base class, except for its constructors and finalizers.</span></span> <span data-ttu-id="810b9-115">Di conseguenza, la classe derivata può riusare il codice definito nella classe di base senza doverlo implementare nuovamente.</span><span class="sxs-lookup"><span data-stu-id="810b9-115">The derived class can thereby reuse the code in the base class without having to re-implement it.</span></span> <span data-ttu-id="810b9-116">Nella classe derivata è possibile aggiungere altri membri.</span><span class="sxs-lookup"><span data-stu-id="810b9-116">In the derived class, you can add more members.</span></span> <span data-ttu-id="810b9-117">In questo modo, la classe derivata estende la funzionalità della classe di base.</span><span class="sxs-lookup"><span data-stu-id="810b9-117">In this manner, the derived class extends the functionality of the base class.</span></span>

## <a name="abstract-base-classes"></a><span data-ttu-id="810b9-137">Classi di base astratte</span><span class="sxs-lookup"><span data-stu-id="810b9-137">Abstract Base Classes</span></span>
