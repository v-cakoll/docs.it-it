# <a name="nullable-types-c-programming-guide"></a><span data-ttu-id="ce4ec-103">Tipi nullable (Guida per programmatori C#)</span><span class="sxs-lookup"><span data-stu-id="ce4ec-103">Nullable types (C# Programming Guide)</span></span>

<span data-ttu-id="ce4ec-104">I tipi nullable sono istanze dello struct <xref:System.Nullable%601?displayProperty=nameWithType>.</span><span class="sxs-lookup"><span data-stu-id="ce4ec-104">Nullable types are instances of the <xref:System.Nullable%601?displayProperty=nameWithType> struct.</span></span> <span data-ttu-id="ce4ec-105">I tipi nullable possono rappresentare tutti i valori di un tipo sottostante `T` e un valore [null](../../language-reference/keywords/null.md) aggiuntivo.</span><span class="sxs-lookup"><span data-stu-id="ce4ec-105">Nullable types can represent all the values of an underlying type `T`, and an additional [null](../../language-reference/keywords/null.md) value.</span></span> <span data-ttu-id="ce4ec-106">Il tipo sottostante `T` può essere qualsiasi [tipo valore](../../language-reference/keywords/value-types.md) non nullable.</span><span class="sxs-lookup"><span data-stu-id="ce4ec-106">The underlying type `T` can be any non-nullable [value type](../../language-reference/keywords/value-types.md).</span></span> <span data-ttu-id="ce4ec-107">`T` non può essere un tipo riferimento.</span><span class="sxs-lookup"><span data-stu-id="ce4ec-107">`T` cannot be a reference type.</span></span>

<span data-ttu-id="ce4ec-108">È ad esempio possibile assegnare `null` o qualsiasi valore integer compreso tra <xref:System.Int32.MinValue?displayProperty=nameWithType> e <xref:System.Int32.MaxValue?displayProperty=nameWithType> a `Nullable<int>` e [true](../../language-reference/keywords/true-literal.md), [false](../../language-reference/keywords/false-literal.md) o `null` a `Nullable<bool>`.</span><span class="sxs-lookup"><span data-stu-id="ce4ec-108">For example, you can assign `null` or any integer value from <xref:System.Int32.MinValue?displayProperty=nameWithType> to <xref:System.Int32.MaxValue?displayProperty=nameWithType> to a `Nullable<int>` and [true](../../language-reference/keywords/true-literal.md), [false](../../language-reference/keywords/false-literal.md), or `null` to a `Nullable<bool>`.</span></span>

<span data-ttu-id="ce4ec-109">Un tipo nullable viene usato quando è necessario rappresentare il valore non definito di un tipo sottostante.</span><span class="sxs-lookup"><span data-stu-id="ce4ec-109">You use a nullable type when you need to represent the undefined value of an underlying type.</span></span> <span data-ttu-id="ce4ec-110">Una variabile booleana può avere solo due valori: true e false.</span><span class="sxs-lookup"><span data-stu-id="ce4ec-110">A Boolean variable can have only two values: true and false.</span></span> <span data-ttu-id="ce4ec-111">Non è previsto alcun valore "non definito".</span><span class="sxs-lookup"><span data-stu-id="ce4ec-111">There is no "undefined" value.</span></span> <span data-ttu-id="ce4ec-112">In molte applicazioni di programmazione, in particolare nelle interazioni tra database, un valore di variabile può essere non definito o mancante.</span><span class="sxs-lookup"><span data-stu-id="ce4ec-112">In many programming applications, most notably database interactions, a variable value can be undefined or missing.</span></span> <span data-ttu-id="ce4ec-113">Ad esempio un campo di un database può contenere i valori true o false oppure è possibile che non contenga alcun valore.</span><span class="sxs-lookup"><span data-stu-id="ce4ec-113">For example, a field in a database may contain the values true or false, or it may contain no value at all.</span></span> <span data-ttu-id="ce4ec-114">In tale caso si usa un tipo `Nullable<bool>`.</span><span class="sxs-lookup"><span data-stu-id="ce4ec-114">You use a `Nullable<bool>` type in that case.</span></span>

<span data-ttu-id="ce4ec-115">I tipi nullable hanno le caratteristiche seguenti:</span><span class="sxs-lookup"><span data-stu-id="ce4ec-115">Nullable types have the following characteristics:</span></span>
  
- <span data-ttu-id="ce4ec-116">I tipi nullable rappresentano variabili di tipo valore a cui può essere assegnato il valore `null`.</span><span class="sxs-lookup"><span data-stu-id="ce4ec-116">Nullable types represent value-type variables that can be assigned the `null` value.</span></span> <span data-ttu-id="ce4ec-117">Non è possibile creare un tipi nullable sulla base di un tipo riferimento.</span><span class="sxs-lookup"><span data-stu-id="ce4ec-117">You cannot create a nullable type based on a reference type.</span></span> <span data-ttu-id="ce4ec-118">I tipi riferimento supportano già il valore `null`.</span><span class="sxs-lookup"><span data-stu-id="ce4ec-118">(Reference types already support the `null` value.)</span></span>  
  
- <span data-ttu-id="ce4ec-119">La sintassi `T?` è un'abbreviazione per `Nullable<T>`.</span><span class="sxs-lookup"><span data-stu-id="ce4ec-119">The syntax `T?` is shorthand for `Nullable<T>`.</span></span> <span data-ttu-id="ce4ec-120">Le due forme sono intercambiabili.</span><span class="sxs-lookup"><span data-stu-id="ce4ec-120">The two forms are interchangeable.</span></span>  
  
- <span data-ttu-id="ce4ec-121">Per assegnare un valore a un tipo nullable è possibile seguire la stessa procedura adottata per un tipo valore sottostante: `int? x = 10;` o `double? d = 4.108;`.</span><span class="sxs-lookup"><span data-stu-id="ce4ec-121">Assign a value to a nullable type just as you would for an underlying value type: `int? x = 10;` or `double? d = 4.108;`.</span></span> <span data-ttu-id="ce4ec-122">È anche possibile assegnare il valore `null`: `int? x = null;`.</span><span class="sxs-lookup"><span data-stu-id="ce4ec-122">You also can assign the `null` value: `int? x = null;`.</span></span>  
  
- <span data-ttu-id="ce4ec-123">Usare le proprietà di sola lettura <xref:System.Nullable%601.HasValue%2A?displayProperty=nameWithType> e <xref:System.Nullable%601.Value%2A?displayProperty=nameWithType> per verificare la presenza di valori null e recuperare il valore, come illustrato nell'esempio seguente: `if (x.HasValue) y = x.Value;`</span><span class="sxs-lookup"><span data-stu-id="ce4ec-123">Use the <xref:System.Nullable%601.HasValue%2A?displayProperty=nameWithType> and <xref:System.Nullable%601.Value%2A?displayProperty=nameWithType> readonly properties to test for null and retrieve the value, as shown in the following example: `if (x.HasValue) y = x.Value;`</span></span>  
  
  - <span data-ttu-id="ce4ec-124">La proprietà <xref:System.Nullable%601.HasValue%2A> restituisce `true` se la variabile contiene un valore o `false` se è `null`.</span><span class="sxs-lookup"><span data-stu-id="ce4ec-124">The <xref:System.Nullable%601.HasValue%2A> property returns `true` if the variable contains a value, or `false` if it's `null`.</span></span>
  
  - <span data-ttu-id="ce4ec-125">La proprietà <xref:System.Nullable%601.Value%2A> restituisce un valore se <xref:System.Nullable%601.HasValue%2A> restituisce `true`.</span><span class="sxs-lookup"><span data-stu-id="ce4ec-125">The <xref:System.Nullable%601.Value%2A> property returns a value if <xref:System.Nullable%601.HasValue%2A> returns `true`.</span></span> <span data-ttu-id="ce4ec-126">In caso contrario, verrà generata un'eccezione <xref:System.InvalidOperationException>.</span><span class="sxs-lookup"><span data-stu-id="ce4ec-126">Otherwise, an <xref:System.InvalidOperationException> is thrown.</span></span>  
  
- <span data-ttu-id="ce4ec-127">Con un tipo nullable è anche possibile usare gli operatori `==` e `!=`, come mostrato nell'esempio seguente: `if (x != null) y = x.Value;`.</span><span class="sxs-lookup"><span data-stu-id="ce4ec-127">You can also use the `==` and `!=` operators with a nullable type, as shown in the following example: `if (x != null) y = x.Value;`.</span></span> <span data-ttu-id="ce4ec-128">Se `a` e `b` sono Null, `a == b` restituisce `true`.</span><span class="sxs-lookup"><span data-stu-id="ce4ec-128">If `a` and `b` are both null, `a == b` evaluates to `true`.</span></span>  

- <span data-ttu-id="ce4ec-129">A partire da C# 7.0, è possibile usare i [criteri di ricerca](../../pattern-matching.md#the-is-type-pattern-expression) per esaminare e ottenere un valore di un tipo nullable: `if (x is int valueOfX) y = valueOfX;`.</span><span class="sxs-lookup"><span data-stu-id="ce4ec-129">Beginning with C# 7.0, you can use [pattern matching](../../pattern-matching.md#the-is-type-pattern-expression) to both examine and get a value of a nullable type: `if (x is int valueOfX) y = valueOfX;`.</span></span>
  
- <span data-ttu-id="ce4ec-130">Il valore predefinito di `T?` è un'istanza la cui proprietà <xref:System.Nullable%601.HasValue%2A> restituisce `false`.</span><span class="sxs-lookup"><span data-stu-id="ce4ec-130">The default value of `T?` is an instance whose <xref:System.Nullable%601.HasValue%2A> property returns `false`.</span></span>  

- <span data-ttu-id="ce4ec-131">Usare il metodo <xref:System.Nullable%601.GetValueOrDefault> per restituire il valore assegnato oppure il valore [predefinito](../../language-reference/keywords/default-values-table.md) del tipo valore sottostante se il valore del tipo nullable è `null`.</span><span class="sxs-lookup"><span data-stu-id="ce4ec-131">Use the <xref:System.Nullable%601.GetValueOrDefault> method to return either the assigned value, or the [default](../../language-reference/keywords/default-values-table.md) value of the underlying value type if the value of the nullable type is `null`.</span></span>  

- <span data-ttu-id="ce4ec-132">Usare il metodo <xref:System.Nullable%601.GetValueOrDefault(%600)> per restituire il valore assegnato oppure il valore predefinito specificato se il valore del tipo nullable è `null`.</span><span class="sxs-lookup"><span data-stu-id="ce4ec-132">Use the <xref:System.Nullable%601.GetValueOrDefault(%600)> method to return either the assigned value, or the provided default value if the value of the nullable type is `null`.</span></span>
  
- <span data-ttu-id="ce4ec-133">Usare l'[operatore di unione Null](../../language-reference/operators/null-coalescing-operator.md), `??`, per assegnare un valore al tipo sottostante in base al valore del tipo nullable: `int? x = null; int y = x ?? -1;`.</span><span class="sxs-lookup"><span data-stu-id="ce4ec-133">Use the [null-coalescing operator](../../language-reference/operators/null-coalescing-operator.md), `??`, to assign a value to an underlying type based on a value of the nullable type: `int? x = null; int y = x ?? -1;`.</span></span> <span data-ttu-id="ce4ec-134">Poiché `x` è Null, nell'esempio il valore restituito di `y` è `-1`.</span><span class="sxs-lookup"><span data-stu-id="ce4ec-134">In the example, since `x` is null, the result value of `y` is `-1`.</span></span>

- <span data-ttu-id="ce4ec-135">Se viene impostata una conversione definita dall'utente tra due tipi di dati, la stessa conversione può essere usata anche con le versioni nullable di questi tipi di dati.</span><span class="sxs-lookup"><span data-stu-id="ce4ec-135">If a user-defined conversion is defined between two data types, the same conversion can also be used with the nullable versions of these data types.</span></span>
  
- <span data-ttu-id="ce4ec-136">I tipi nullable nidificati non sono consentiti.</span><span class="sxs-lookup"><span data-stu-id="ce4ec-136">Nested nullable types are not allowed.</span></span> <span data-ttu-id="ce4ec-137">La riga seguente non verrà compilata: `Nullable<Nullable<int>> n;`</span><span class="sxs-lookup"><span data-stu-id="ce4ec-137">The following line doesn't compile: `Nullable<Nullable<int>> n;`</span></span>
