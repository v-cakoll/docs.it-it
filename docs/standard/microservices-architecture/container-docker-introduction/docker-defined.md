- <span data-ttu-id="84283-121">I contenitori Hyper-V espandono l'isolamento fornito dai contenitori Windows Server tramite l'esecuzione di ogni contenitore in una macchina virtuale con ottimizzazione elevata.</span><span class="sxs-lookup"><span data-stu-id="84283-121">Hyper-V Containers expand on the isolation provided by Windows Server Containers by running each container in a highly optimized virtual machine.</span></span> <span data-ttu-id="84283-122">In questa configurazione, il kernel dell'host dei contenitori non è condiviso con i contenitori Hyper-V per fornire un isolamento migliore.</span><span class="sxs-lookup"><span data-stu-id="84283-122">In this configuration, the kernel of the container host isn't shared with the Hyper-V Containers, providing better isolation.</span></span>

L'esecuzione di un contenitore Hyper-V richiede un parametro aggiuntivo.</span><span class="sxs-lookup"><span data-stu-id="84283-124">The difference is in how the container is created from the image running a Hyper-V Container requires an extra parameter.</span></span> <span data-ttu-id="84283-125">Per informazioni dettagliate, vedere [Contenitori Hyper-V](https://docs.microsoft.com/virtualization/windowscontainers/manage-containers/hyperv-container).</span><span class="sxs-lookup"><span data-stu-id="84283-125">For details, see [Hyper-V Containers](https://docs.microsoft.com/virtualization/windowscontainers/manage-containers/hyperv-container).</span></span>

## <a name="a-simple-analogy"></a><span data-ttu-id="84283-148">Una semplice analogia</span><span class="sxs-lookup"><span data-stu-id="84283-148">A simple analogy</span></span>

<span data-ttu-id="84283-149">Forse una semplice analogia può aiutare a comprendere a fondo il concetto di base di Docker.</span><span class="sxs-lookup"><span data-stu-id="84283-149">Perhaps a simple analogy can help getting the grasp of the core concept of Docker.</span></span>

<span data-ttu-id="84283-150">Torniamo per qualche istante agli anni '50.</span><span class="sxs-lookup"><span data-stu-id="84283-150">Let's go back in time to the 1950s for a moment.</span></span> <span data-ttu-id="84283-151">Non esistevano elaboratori di testo e quasi ovunque venivano usate le fotocopiatrici.</span><span class="sxs-lookup"><span data-stu-id="84283-151">There were no word processors, and the photocopiers were used everywhere (kind of).</span></span>

<span data-ttu-id="84283-152">Si immagini di dover distribuire rapidamente un numero elevato di lettere, ovvero di spedirle ai clienti, usando carta e buste, e di recapitarle fisicamente all'indirizzo di ogni cliente perché la posta elettronica non esisteva.</span><span class="sxs-lookup"><span data-stu-id="84283-152">Imagine you're responsible for quickly issuing batches of letters as required, to mail them to customers, using real paper and envelopes, to be delivered physically to each customer's address (there was no email back then).</span></span>

<span data-ttu-id="84283-153">A un certo punto ci si rende conto che le lettere sono semplicemente composte da una lunga serie di paragrafi, che vengono presi e combinati in base alle esigenze e allo scopo della lettera e pertanto si progetta un sistema per distribuire le lettere rapidamente, aspettandosi un incremento notevole.</span><span class="sxs-lookup"><span data-stu-id="84283-153">At some point, you realize the letters are just a composition of a large set of paragraphs, which are picked and arranged as needed, according to the purpose of the letter, so you devise a system to issue letters quickly, expecting to get a hefty raise.</span></span>

<span data-ttu-id="84283-154">Il sistema è semplice:</span><span class="sxs-lookup"><span data-stu-id="84283-154">The system is simple:</span></span>

1. <span data-ttu-id="84283-155">Si inizia con un blocco di fogli trasparenti ognuno dei quali contiene un paragrafo.</span><span class="sxs-lookup"><span data-stu-id="84283-155">You begin with a deck of transparent sheets containing one paragraph each.</span></span>

2. <span data-ttu-id="84283-156">Per distribuire una serie di lettere, si prendono i fogli con i paragrafi necessari, quindi li si dispone e li si allinea in modo da poterli leggere agevolmente.</span><span class="sxs-lookup"><span data-stu-id="84283-156">To issue a set of letters, you pick the sheets with the paragraphs you need, then you stack and align them so they look and read fine.</span></span>

3. <span data-ttu-id="84283-157">Infine, si inserisce il blocco nella fotocopiatrice e si premere Start per ottenere tutte le lettere necessarie.</span><span class="sxs-lookup"><span data-stu-id="84283-157">Finally, you place the set in the photocopier and press start to produce as many letters as required.</span></span>
