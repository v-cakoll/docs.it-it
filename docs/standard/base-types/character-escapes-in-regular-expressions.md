-   <span data-ttu-id="b1818-104">Il carattere che segue è un carattere speciale, come illustrato nella tabella nella sezione seguente.</span><span class="sxs-lookup"><span data-stu-id="b1818-104">The character that follows it is a special character, as shown in the table in the following section.</span></span> <span data-ttu-id="b1818-105">Ad esempio, `\b` è un ancoraggio che indica che una corrispondenza di un'espressione regolare deve iniziare sul confine di una parola, `\t` rappresenta un carattere di tabulazione e `\x020` rappresenta uno spazio.</span><span class="sxs-lookup"><span data-stu-id="b1818-105">For example, `\b` is an anchor that indicates that a regular expression match should begin on a word boundary, `\t` represents a tab, and `\x020` represents a space.</span></span>

> [!NOTE]
