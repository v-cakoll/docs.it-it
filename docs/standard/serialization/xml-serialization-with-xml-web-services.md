### <a name="example-of-using-both-styles"></a><span data-ttu-id="22ace-113">Esempio di utilizzo di entrambi gli stili</span><span class="sxs-lookup"><span data-stu-id="22ace-113">Example of Using Both Styles</span></span>  
 <span data-ttu-id="22ace-114">Durante la creazione di un servizio Web XML, è possibile utilizzare entrambi gli insiemi di attributi sui metodi.</span><span class="sxs-lookup"><span data-stu-id="22ace-114">When you're creating an XML Web service, you can use both sets of attributes on the methods.</span></span> <span data-ttu-id="22ace-115">Nell'esempio di codice riportato di seguito, la classe denominata `MyService` contiene due metodi del servizio Web XML, `MyLiteralMethod` e `MyEncodedMethod`.</span><span class="sxs-lookup"><span data-stu-id="22ace-115">In the following code example, the class named `MyService` contains two XML Web service methods, `MyLiteralMethod` and `MyEncodedMethod`.</span></span> <span data-ttu-id="22ace-116">Entrambi i metodi eseguono la stessa funzione: la restituzione di un'istanza della classe `Order`.</span><span class="sxs-lookup"><span data-stu-id="22ace-116">Both methods perform the same function: returning an instance of the `Order` class.</span></span> <span data-ttu-id="22ace-117">Nella classe `Order` gli attributi <xref:System.Xml.Serialization.XmlTypeAttribute> e <xref:System.Xml.Serialization.SoapTypeAttribute> sono entrambi applicati al campo `OrderID` ed entrambi hanno la proprietà `ElementName` impostata su valori diversi.</span><span class="sxs-lookup"><span data-stu-id="22ace-117">In the `Order` class, the <xref:System.Xml.Serialization.XmlTypeAttribute> and the <xref:System.Xml.Serialization.SoapTypeAttribute> attributes are both applied to the `OrderID` field, and both attributes have their `ElementName` property set to different values.</span></span>  
  
 <span data-ttu-id="22ace-118">Per eseguire l'esempio, incollare il codice in un file con estensione asmx e posizionare il file in una directory virtuale gestita da Internet Information Services (IIS).</span><span class="sxs-lookup"><span data-stu-id="22ace-118">To run the example, paste the code into a file with an .asmx extension, and place the file into a virtual directory managed by Internet Information Services (IIS).</span></span> <span data-ttu-id="22ace-119">Da un browser HTML, ad esempio Internet Explorer, digitare il nome del computer, della directory virtuale e del file.</span><span class="sxs-lookup"><span data-stu-id="22ace-119">From an HTML browser, such as Internet Explorer, type the name of the computer, virtual directory, and file.</span></span>  
  
```vb  
<%@ WebService Language="VB" Class="MyService" %>  
Imports System  
Imports System.Web.Services  
Imports System.Web.Services.Protocols  
Imports System.Xml.Serialization  
Public Class Order  
    ' Both types of attributes can be applied. Depending on which type  
    ' the method used, either one will affect the call.  
    <SoapElement(ElementName:= "EncodedOrderID"), _  
    XmlElement(ElementName:= "LiteralOrderID")> _  
    public OrderID As String  
End Class  
  
Public Class MyService  
    <WebMethod, SoapDocumentMethod> _  
    public Function MyLiteralMethod() As Order   
        Dim myOrder As Order = New Order()  
        return myOrder  
    End Function  
    <WebMethod, SoapRpcMethod> _  
    public Function MyEncodedMethod() As Order   
        Dim myOrder As Order = New Order()  
        return myOrder  
    End Function  
End Class  
```  
  
```csharp  
<%@ WebService Language="C#" Class="MyService" %>  
using System;  
using System.Web.Services;  
using System.Web.Services.Protocols;  
using System.Xml.Serialization;  
public class Order{  
    // Both types of attributes can be applied. Depending on which type  
    // the method used, either one will affect the call.  
    [SoapElement(ElementName = "EncodedOrderID")]  
    [XmlElement(ElementName = "LiteralOrderID")]  
    public String OrderID;  
}  
public class MyService{  
    [WebMethod][SoapDocumentMethod]  
    public Order MyLiteralMethod(){  
        Order myOrder = new Order();  
        return myOrder;  
    }  
    [WebMethod][SoapRpcMethod]  
    public Order MyEncodedMethod(){  
        Order myOrder = new Order();  
        return myOrder;  
    }  
}  
```  
  
 <span data-ttu-id="22ace-120">Nell'esempio di codice riportato di seguito viene chiamato il metodo `MyLiteralMethod`.</span><span class="sxs-lookup"><span data-stu-id="22ace-120">The following code example calls `MyLiteralMethod`.</span></span> <span data-ttu-id="22ace-121">Il nome dell'elemento viene modificato in "LiteralOrderID."</span><span class="sxs-lookup"><span data-stu-id="22ace-121">The element name is changed to "LiteralOrderID".</span></span>  
  
```xml  
<?xml version="1.0" encoding="utf-8"?>

    <soap:Body>  
        <MyLiteralMethodResponse xmlns="http://tempuri.org/">  
            <MyLiteralMethodResult>  
                <LiteralOrderID>string</LiteralOrderID>  
            </MyLiteralMethodResult>  
        </MyLiteralMethodResponse>  
    </soap:Body>  
</soap:Envelope>  
```  
  
 <span data-ttu-id="22ace-122">Nell'esempio di codice riportato di seguito viene chiamato il metodo `MyEncodedMethod`.</span><span class="sxs-lookup"><span data-stu-id="22ace-122">The following code example calls `MyEncodedMethod`.</span></span> <span data-ttu-id="22ace-123">Il nome dell'elemento è "EncodedOrderID."</span><span class="sxs-lookup"><span data-stu-id="22ace-123">The element name is "EncodedOrderID".</span></span>  
  
