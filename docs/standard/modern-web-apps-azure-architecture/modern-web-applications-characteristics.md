<span data-ttu-id="a9b78-150">Molte applicazioni Web presentano una combinazione di comportamenti dell'applicazione Web tradizionale (in genere per il contenuto) e applicazioni a pagina singola (per l'interattività).</span><span class="sxs-lookup"><span data-stu-id="a9b78-150">Many web applications involve a combination of traditional web application behavior (typically for content) and SPAs (for interactivity).</span></span> <span data-ttu-id="a9b78-151">ASP.NET Core supporta sia i framework MVC sia le API Web nella stessa applicazione, usando lo stesso set di strumenti e librerie del framework sottostanti.</span><span class="sxs-lookup"><span data-stu-id="a9b78-151">ASP.NET Core supports both MVC and web APIs in the same application, using the same set of tools and underlying framework libraries.</span></span>
