data-ttu-id="70045-107">Non è possibile eseguire l'override delle proprietà ereditate in un tipo derivato.</span><span class="sxs-lookup"><span data-stu-id="70045-107">You cannot override inherited  <span data-ttu-id="70045-108">In un modello concettuale è possibile compilare gerarchie di ereditarietà nelle quali un tipo derivato eredita da un altro tipo derivato.</span><span class="sxs-lookup"><span data-stu-id="70045-108">In a conceptual model you can build inheritance hierarchies in which a derived type inherits from another derived type.</span></span> <span data-ttu-id="70045-109">Il tipo nella parte superiore della gerarchia (il tipo nella gerarchia che non è un tipo derivato) viene chiamato il *tipo radice*.</span><span class="sxs-lookup"><span data-stu-id="70045-109">The type at the top of the hierarchy (the one type in the hierarchy that is not a derived type) is called the *root type*.</span></span> <span data-ttu-id="70045-110">In una gerarchia di ereditarietà, la [chiave di entità](../../../../docs/framework/data/adonet/entity-key.md) devono essere definiti sul tipo radice.</span><span class="sxs-lookup"><span data-stu-id="70045-110">In an inheritance hierarchy, the [entity key](../../../../docs/framework/data/adonet/entity-key.md) must be defined on the root type.</span></span>  
  
 <span data-ttu-id="70045-111">Non è possibile compilare gerarchie di ereditarietà nelle quali un tipo derivato eredita da più di un tipo.</span><span class="sxs-lookup"><span data-stu-id="70045-111">You cannot build inheritance hierarchies in which a derived type inherits from more than one type.</span></span> <span data-ttu-id="70045-112">In un modello concettuale con un tipo di entità `Book`, ad esempio, è possibile definire i tipi derivati `FictionBook` e `NonFictionBook` che ereditano ognuno da `Book`.</span><span class="sxs-lookup"><span data-stu-id="70045-112">For example, in a conceptual model with a `Book` entity type, you could define derived types `FictionBook` and `NonFictionBook` that each inherit from `Book`.</span></span> <span data-ttu-id="70045-113">Non è quindi possibile, tuttavia, definire un tipo che eredita sia dal tipo `FictionBook` che dal tipo `NonFictionBook`.</span><span class="sxs-lookup"><span data-stu-id="70045-113">However, you could not then define a type that inherits from both the `FictionBook` and `NonFictionBook` types.</span></span>
