### <a name="the-connectionstrings-section"></a><span data-ttu-id="c37a9-111">Sezione connectionStrings</span><span class="sxs-lookup"><span data-stu-id="c37a9-111">The connectionStrings Section</span></span>

    <connectionStrings>

    </connectionStrings>  
  </configuration>
