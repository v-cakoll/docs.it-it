usato.</span><span class="sxs-lookup"><span data-stu-id="672b4-105">When called from a querying technology, such as LINQ to Entities, these canonical functions are translated to the correct corresponding store function for the provider being used.</span></span> <span data-ttu-id="672b4-106">Le chiamate alle funzioni possono pertanto essere espresse con una forma comune indipendentemente
