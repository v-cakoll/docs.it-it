## <a name="scaling-in-windows-vista"></a><span data-ttu-id="3febd-111">Ridimensionamento in Windows Vista</span><span class="sxs-lookup"><span data-stu-id="3febd-111">Scaling in Windows Vista</span></span>

 <span data-ttu-id="3febd-115">La modifica dell'impostazione DPI non corrisponde alla modifica della risoluzione dello schermo.</span><span class="sxs-lookup"><span data-stu-id="3febd-115">Changing the dpi setting is not the same as changing the screen resolution.</span></span> <span data-ttu-id="3febd-116">Con il ridimensionamento DPI, il numero di pixel fisici sullo schermo rimane lo stesso.</span><span class="sxs-lookup"><span data-stu-id="3febd-116">With dpi scaling, the number of physical pixels on the screen remains the same.</span></span> <span data-ttu-id="3febd-117">Il ridimensionamento viene tuttavia applicato alle dimensioni e alla posizione degli elementi dell' [!INCLUDE[TLA2#tla_ui](../../../includes/tla2sharptla-ui-md.md)] .</span><span class="sxs-lookup"><span data-stu-id="3febd-117">However, scaling is applied to the size and location of [!INCLUDE[TLA2#tla_ui](../../../includes/tla2sharptla-ui-md.md)] elements.</span></span> <span data-ttu-id="3febd-118">È possibile eseguire il ridimensionamento automaticamente con Gestione finestre desktop per il desktop e per le applicazioni che non richiedono tale funzionalità in modo esplicito.</span><span class="sxs-lookup"><span data-stu-id="3febd-118">This scaling can be performed automatically by the Desktop Window Manager (DWM) for the desktop and for applications that do not explicitly ask not to be scaled.</span></span>

> [!NOTE]
> <span data-ttu-id="3febd-123">Per impostazione predefinita, DWM non esegue la scalabilità per le applicazioni non compatibili con DPI quando l'utente imposta il valore dpi su 120, ma lo esegue quando il valore dpi è impostato su un valore personalizzato pari a 144 o superiore.</span><span class="sxs-lookup"><span data-stu-id="3febd-123">By default, the DWM does not perform scaling for non-dpi-aware applications when the user sets the dpi to 120, but does perform it when the dpi is set to a custom value of 144 or higher.</span></span> <span data-ttu-id="3febd-124">È tuttavia possibile ignorare questo comportamento predefinito.</span><span class="sxs-lookup"><span data-stu-id="3febd-124">However, the user can override the default behavior.</span></span>  
  
 <span data-ttu-id="3febd-125">Il ridimensionamento dello schermo crea nuove sfide per le applicazioni che dipendono dalle coordinate dello schermo.</span><span class="sxs-lookup"><span data-stu-id="3febd-125">Screen scaling creates new challenges for applications that are concerned in any way with screen coordinates.</span></span> <span data-ttu-id="3febd-126">Lo schermo contiene ora due sistemi di coordinate: fisico e logico.</span><span class="sxs-lookup"><span data-stu-id="3febd-126">The screen now contains two coordinate systems: physical and logical.</span></span> <span data-ttu-id="3febd-127">Le coordinate fisiche di un punto sono l'offset effettivo in pixel dalla parte superiore sinistra dell'origine.</span><span class="sxs-lookup"><span data-stu-id="3febd-127">The physical coordinates of a point are the actual offset in pixels from the top left of the origin.</span></span> <span data-ttu-id="3febd-128">Le coordinate logiche sono gli offset così come si presenterebbero se i pixel fossero ridimensionati.</span><span class="sxs-lookup"><span data-stu-id="3febd-128">The logical coordinates are the offsets as they would be if the pixels themselves were scaled.</span></span>  
  
 <span data-ttu-id="3febd-129">Si supponga di progettare una finestra di dialogo con un pulsante in corrispondenza delle coordinate (100, 48).</span><span class="sxs-lookup"><span data-stu-id="3febd-129">Suppose you design a dialog box with a button at coordinates (100, 48).</span></span> <span data-ttu-id="3febd-130">Quando questa finestra di dialogo viene visualizzata sul valore predefinito 96 dpi, il pulsante si trova in corrispondenza delle coordinate fisiche (100, 48).</span><span class="sxs-lookup"><span data-stu-id="3febd-130">When this dialog box is displayed at the default 96 dpi, the button is located at physical coordinates of (100, 48).</span></span> <span data-ttu-id="3febd-131">A 120 dpi si trova in corrispondenza delle coordinate fisiche (125, 60).</span><span class="sxs-lookup"><span data-stu-id="3febd-131">At 120 dpi, it is located at physical coordinates of (125, 60).</span></span> <span data-ttu-id="3febd-132">Tuttavia, le coordinate logiche sono le stesse in qualsiasi impostazione dpi: (100, 48).</span><span class="sxs-lookup"><span data-stu-id="3febd-132">But the logical coordinates are the same at any dpi setting: (100, 48).</span></span>  
  
 <span data-ttu-id="3febd-133">Le coordinate logiche sono importanti, perché rendono il comportamento del sistema operativo e delle applicazioni coerenti indipendentemente dall'impostazione dpi.</span><span class="sxs-lookup"><span data-stu-id="3febd-133">Logical coordinates are important, because they make the behavior of the operating system and applications consistent regardless of the dpi setting.</span></span> <span data-ttu-id="3febd-134">Ad esempio, la proprietà <xref:System.Windows.Forms.Cursor.Position%2A?displayProperty=nameWithType> restituisce in genere le coordinate logiche.</span><span class="sxs-lookup"><span data-stu-id="3febd-134">For example, <xref:System.Windows.Forms.Cursor.Position%2A?displayProperty=nameWithType> normally returns the logical coordinates.</span></span> <span data-ttu-id="3febd-135">Se si sposta il cursore su un elemento in una finestra di dialogo, le stesse coordinate vengono restituite indipendentemente dall'impostazione dpi.</span><span class="sxs-lookup"><span data-stu-id="3febd-135">If you move the cursor over an element in a dialog box, the same coordinates are returned regardless of the dpi setting.</span></span> <span data-ttu-id="3febd-136">Se si disegna un controllo in (100, 100), questo viene disegnato su queste coordinate logiche e occupa la stessa posizione relativa in qualsiasi impostazione dpi.</span><span class="sxs-lookup"><span data-stu-id="3febd-136">If you draw a control at (100, 100), it is drawn to those logical coordinates, and will occupy the same relative position at any dpi setting.</span></span>
