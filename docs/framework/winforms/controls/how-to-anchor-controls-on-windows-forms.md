### <a name="to-anchor-a-control-on-a-form"></a><span data-ttu-id="69d76-115">Per ancorare un controllo in un form</span><span class="sxs-lookup"><span data-stu-id="69d76-115">To anchor a control on a form</span></span>  
  
1.  <span data-ttu-id="69d76-116">Selezionare il controllo che si desidera ancorare.</span><span class="sxs-lookup"><span data-stu-id="69d76-116">Select the control you want to anchor.</span></span>  
  
    > [!NOTE]
    >  <span data-ttu-id="69d76-117">È possibile ancorare più controlli contemporaneamente tenendo premuto CTRL, facendo clic su ogni controllo per selezionarla e quindi seguendo il resto di questa procedura.</span><span class="sxs-lookup"><span data-stu-id="69d76-117">You can anchor multiple controls simultaneously by pressing the CTRL key, clicking each control to select it, and then following the rest of this procedure.</span></span>

     <span data-ttu-id="69d76-119">Viene visualizzato un editor che mostra una croce.</span><span class="sxs-lookup"><span data-stu-id="69d76-119">An editor is displayed that shows a cross.</span></span>  
  
3.  <span data-ttu-id="69d76-120">Per impostare un punto di ancoraggio, fare clic su alto, a sinistra, destra, ovvero nella sezione inferiore della finestra di tra.</span><span class="sxs-lookup"><span data-stu-id="69d76-120">To set an anchor, click the top, left, right, or bottom section of the cross.</span></span>  
  
     <span data-ttu-id="69d76-121">I controlli ancorati nella parte superiore e sinistra per impostazione predefinita.</span><span class="sxs-lookup"><span data-stu-id="69d76-121">Controls are anchored to the top and left by default.</span></span>  
  
4.  <span data-ttu-id="69d76-122">Per cancellare un lato del controllo che è stato ancorato, fare clic su tale parte della croce.</span><span class="sxs-lookup"><span data-stu-id="69d76-122">To clear a side of the control that has been anchored, click that arm of the cross.</span></span>  
  
5.  <span data-ttu-id="69d76-123">Per chiudere la <xref:System.Windows.Forms.Control.Anchor%2A> editor di proprietà, fare clic su di <xref:System.Windows.Forms.Control.Anchor%2A> nuovo nome della proprietà.</span><span class="sxs-lookup"><span data-stu-id="69d76-123">To close the <xref:System.Windows.Forms.Control.Anchor%2A> property editor, click the <xref:System.Windows.Forms.Control.Anchor%2A> property name again.</span></span>  
  
 <span data-ttu-id="69d76-124">Quando il form viene visualizzato in fase di esecuzione, il controllo viene ridimensionato per rimanere posizionato alla stessa distanza dal bordo del form.</span><span class="sxs-lookup"><span data-stu-id="69d76-124">When your form is displayed at run time, the control resizes to remain positioned at the same distance from the edge of the form.</span></span> <span data-ttu-id="69d76-125">La distanza dal bordo ancorato rimane sempre lo stesso come distanza definito quando viene posizionato il controllo in Progettazione Windows Form.</span><span class="sxs-lookup"><span data-stu-id="69d76-125">The distance from the anchored edge always remains the same as the distance defined when the control is positioned in the Windows Forms Designer.</span></span>  
  
> [!NOTE]
>  <span data-ttu-id="69d76-126">Alcuni controlli, ad esempio il <xref:System.Windows.Forms.ComboBox> , dispongono di un limite all'altezza.</span><span class="sxs-lookup"><span data-stu-id="69d76-126">Certain controls, such as the <xref:System.Windows.Forms.ComboBox> control, have a limit to their height.</span></span> <span data-ttu-id="69d76-127">Il controllo nella parte inferiore del form o contenitore di ancoraggio non è possibile forzare il controllo di superare il limite di altezza.</span><span class="sxs-lookup"><span data-stu-id="69d76-127">Anchoring the control to the bottom of its form or container cannot force the control to exceed its height limit.</span></span>  
  
 <span data-ttu-id="69d76-128">I controlli ereditati devono essere `Protected` per poter essere ancorata.</span><span class="sxs-lookup"><span data-stu-id="69d76-128">Inherited controls must be `Protected` to be able to be anchored.</span></span> <span data-ttu-id="69d76-129">Per modificare il livello di accesso di un controllo, impostare il relativo `Modifiers` proprietà nel **proprietà** finestra.</span><span class="sxs-lookup"><span data-stu-id="69d76-129">To change the access level of a control, set its `Modifiers` property in the **Properties** window.</span></span>
