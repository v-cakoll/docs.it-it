## <a name="criteria-for-using-a-wf-3-activity-with-an-interop-activity"></a><span data-ttu-id="17023-107">Criteri per l'uso di un'attività di WF 3 con un'attività di interoperabilità</span><span class="sxs-lookup"><span data-stu-id="17023-107">Criteria for Using a WF 3 Activity with an Interop Activity</span></span>  
 <span data-ttu-id="17023-108">Affinché un'attività di WF 3 venga eseguita correttamente all'interno di un'attività <xref:System.Activities.Statements.Interop>, devono essere soddisfatti i seguenti criteri:</span><span class="sxs-lookup"><span data-stu-id="17023-108">For a WF 3 activity to successfully execute within an <xref:System.Activities.Statements.Interop> activity, the following criteria must be met:</span></span>
