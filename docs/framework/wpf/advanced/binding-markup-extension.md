## <a name="binding-expression-usage"></a><span data-ttu-id="86a67-104">Utilizzo dell'espressione di associazione</span><span class="sxs-lookup"><span data-stu-id="86a67-104">Binding Expression Usage</span></span>  
  
```  
<object property="{Binding}" .../>  
-or-  
<object property="{Binding  bindProp1=value1[, bindPropN=valueN]*}" ...  
/>  
-or-  
<object property="{Binding path}" .../>  
-or  
<object property="{Binding path[, bindPropN=valueN]*}" .../>  
```
