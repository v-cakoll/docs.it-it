# <a name="icorprofilercallbackruntimethreadresumed-method"></a>Metodo ICorProfilerCallback::RuntimeThreadResumed

## <a name="syntax"></a>Sintassi  
  
```  
HRESULT RuntimeThreadResumed(  
    [in] ThreadID threadId);  
```
