# <a name="icordebugassembly2-interface"></a>Interfaccia ICorDebugAssembly2

Rappresenta un assembly. Questa interfaccia è un'estensione dell'interfaccia ICorDebugAssembly.  
  
## <a name="methods"></a>Metodi  
  
|Metodo|Descrizione|  
|------------|-----------------|  
|[Metodo IsFullyTrusted](../../../../docs/framework/unmanaged-api/debugging/icordebugassembly2-isfullytrusted-method.md)|Ottiene un valore che indica se l'assembly è stata concessa l'attendibilità totale da sistema di sicurezza di runtime.|  
  
## <a name="remarks"></a>Note  
  
> [!NOTE]
>  Questa interfaccia non supporta la chiamata in modalità remota, tra computer o tra processi.
