# <a name="systemservicemodeltxsourcetxscoperequiredisattachedtransaction"></a>System.ServiceModel.TxSourceTxScopeRequiredIsAttachedTransaction
È stata ricevuta la transazione specificata per l'operazione specificata da una transazione InstanceContext.  
  
## <a name="description"></a>Descrizione  
 Viene tracciato se la transazione disponibile per il codice utente è stata tralasciata da un metodo AutoComplete=false chiamato prima di questa chiamata.
