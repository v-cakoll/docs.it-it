# <a name="messagequeueduplicatedpipeleak"></a>MessageQueueDuplicatedPipeLeak
Id: 166  
  
 Gravità: Error  
  
 Categoria: SMSvcHost  
  
## <a name="description"></a>Descrizione  
 Questo evento indica che si è verificato un errore durante l'invio di una pipe con nome duplicata. L'handle è perso nel processo. Vengono indicati il l'origine, l'eccezione, il nome e l'ID del processo.
