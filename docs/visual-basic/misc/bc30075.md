# <a name="named-arguments-are-not-valid-as-array-subscripts"></a>Gli argomenti denominati non sono validi come indici di matrice
Un indice di matrice viene specificato usando la sintassi per passare un argomento di routine in base al nome, ad esempio `Array(Index := 10)`. Questa sintassi non Ã¨ valida per gli indici di matrice.  
  
 **ID errore:** BC30075  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore
