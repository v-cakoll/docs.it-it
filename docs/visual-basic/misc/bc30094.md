# <a name="label-39ltlabelnamegt39-is-already-defined-in-the-current-methodmultiline-lambda"></a>Etichetta &#39; &lt;NomeEtichetta&gt; &#39; è già definita nel corrente metodo o nell'espressione lambda
Il nome dell'etichetta viene duplicato all'interno della routine. Tutti i nomi all'interno di una routine devono essere univoci.  
  
 **ID errore:** BC30094  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Rinominare l'etichetta duplicata.
