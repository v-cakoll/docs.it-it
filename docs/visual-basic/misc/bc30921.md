# <a name="39ltderivedtypenamegt39-cannot-inherit-from-lttypegt-39ltconstructedbasetypenamegt39-because-it-expands-the-access-of-type-39ltinternaltypenamegt39-to-ltregiongt-39ltregionnamegt39"></a><span data-ttu-id="6ca9f-102">&#39; &lt;nometipoderivato&gt;&#39; non può ereditare da &lt;tipo&gt; &#39;&lt; nometipobasecreato&gt;&#39; perché espande l'accesso di tipo &#39;&lt; nometipointerno&gt;&#39; a &lt;area&gt; &#39;&lt; RegionName&gt;&#39;</span><span class="sxs-lookup"><span data-stu-id="6ca9f-102">&#39;&lt;derivedtypename&gt;&#39; cannot inherit from &lt;type&gt; &#39;&lt;constructedbasetypename&gt;&#39; because it expands the access of type &#39;&lt;internaltypename&gt;&#39; to &lt;region&gt; &#39;&lt;regionname&gt;&#39;</span></span>
<span data-ttu-id="6ca9f-103">Una classe o un'interfaccia derivata prova a espandere il livello di accesso di un tipo interno usandolo come argomento di tipo per una classe o un'interfaccia base.</span><span class="sxs-lookup"><span data-stu-id="6ca9f-103">A derived class or interface attempts to expand the access level of an internal type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="6ca9f-104">Il codice seguente può generare questo errore.</span><span class="sxs-lookup"><span data-stu-id="6ca9f-104">The following code can generate this error.</span></span>  
  
```  
Public Class containingClass  
    Public Class baseClass(Of t)  
    End Class  
    Friend Class derivedClass  
        Inherits baseClass(Of internalStructure)  
    End Class  
    Private Structure internalStructure  
        Dim firstMember As Integer  
    End Structure  
End Class  
```  
  
 <span data-ttu-id="6ca9f-105">Al codice esterno a `containingClass` non è consentito l'accesso a `internalStructure`.</span><span class="sxs-lookup"><span data-stu-id="6ca9f-105">Code outside `containingClass` is not allowed to access `internalStructure`.</span></span> <span data-ttu-id="6ca9f-106">Tuttavia, è possibile accedere a `derivedClass` da qualsiasi codice nello stesso assembly.</span><span class="sxs-lookup"><span data-stu-id="6ca9f-106">However, `derivedClass` can be accessed from any code in the same assembly.</span></span> <span data-ttu-id="6ca9f-107">Per questo motivo `derivedClass` non può ereditare `baseClass` se usa `internalStructure` come argomento di tipo, perché ciò potrebbe esporre `internalStructure` in tutta la regione del codice di definizione.</span><span class="sxs-lookup"><span data-stu-id="6ca9f-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `internalStructure` as a type argument, because that could expose `internalStructure` throughout the defining code region.</span></span>  
  
 <span data-ttu-id="6ca9f-108">**ID errore:** BC30921</span><span class="sxs-lookup"><span data-stu-id="6ca9f-108">**Error ID:** BC30921</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="6ca9f-109">Per correggere l'errore</span><span class="sxs-lookup"><span data-stu-id="6ca9f-109">To correct this error</span></span>  
  
-   <span data-ttu-id="6ca9f-110">Regolare i livelli di accesso delle classi o delle interfacce in modo che il tipo derivato non espanda il livello di accesso del tipo interno.</span><span class="sxs-lookup"><span data-stu-id="6ca9f-110">Adjust the access levels of the classes or interfaces so that the derived type does not expand the access level of the internal type.</span></span>  
  
     <span data-ttu-id="6ca9f-111">-oppure-</span><span class="sxs-lookup"><span data-stu-id="6ca9f-111">-or-</span></span>  
  
-   <span data-ttu-id="6ca9f-112">Se non è possibile regolare i livelli di accesso, non usare il tipo interno come argomento di tipo durante la costruzione della classe o dell'interfaccia base.</span><span class="sxs-lookup"><span data-stu-id="6ca9f-112">If you cannot adjust the access levels, then do not use the internal type as a type argument when constructing the base class or interface.</span></span>
