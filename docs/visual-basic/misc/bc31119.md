# <a name="39end-removehandler39-must-be-the-first-statement-on-a-line"></a>&#39;End RemoveHandler&#39; deve essere la prima istruzione in una riga
Un'istruzione `End RemoveHandler` segue un separatore di istruzioni due punti (:). `End RemoveHandler` deve essere l'unica istruzione nella riga di origine.  
  
 **ID errore:** BC31119
