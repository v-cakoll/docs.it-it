# <a name="39readonly39-and-39writeonly39-cannot-be-combined"></a>&#39;Sola lettura&#39; e &#39;WriteOnly&#39; non possono essere combinati
I modificatori `ReadOnly` e `WriteOnly` si escludono reciprocamente.  
  
 **ID errore:** BC30635  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Specificare `ReadOnly` o `WriteOnly`, ma non entrambi.
