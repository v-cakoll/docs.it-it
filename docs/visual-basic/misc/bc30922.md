# <a name="39ltderivedtypenamegt39-cannot-inherit-from-lttypegt-39ltconstructedbasetypenamegt39-because-it-expands-the-access-of-type-39ltinternaltypenamegt39-outside-the-assembly"></a><span data-ttu-id="7265c-102">&#39; &lt;nometipoderivato&gt;&#39; non può ereditare da &lt;tipo&gt; &#39;&lt; nometipobasecreato&gt;&#39; perché espande l'accesso di tipo &#39;&lt; nometipointerno&gt;&#39; all'esterno dell'assembly</span><span class="sxs-lookup"><span data-stu-id="7265c-102">&#39;&lt;derivedtypename&gt;&#39; cannot inherit from &lt;type&gt; &#39;&lt;constructedbasetypename&gt;&#39; because it expands the access of type &#39;&lt;internaltypename&gt;&#39; outside the assembly</span></span>
<span data-ttu-id="7265c-103">Una classe o un'interfaccia derivata prova a espandere il livello di accesso di un tipo ristretto usandolo come argomento di tipo per una classe o un'interfaccia base.</span><span class="sxs-lookup"><span data-stu-id="7265c-103">A derived class or interface attempts to expand the access level of a restricted type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="7265c-104">Il codice seguente può generare questo errore.</span><span class="sxs-lookup"><span data-stu-id="7265c-104">The following code can generate this error.</span></span>  
  
```  
Public Class baseClass(Of t)  
End Class  
Public Class derivedClass  
    Inherits baseClass(Of restrictedStructure)  
End Class  
Friend Structure restrictedStructure  
    Dim firstMember As Integer  
End Structure  
```  
  
 <span data-ttu-id="7265c-105">Il codice all'esterno dell'assembly non può accedere a `restrictedStructure`.</span><span class="sxs-lookup"><span data-stu-id="7265c-105">Code outside the assembly is not allowed to access `restrictedStructure`.</span></span> <span data-ttu-id="7265c-106">L'accesso all'oggetto `derivedClass` , tuttavia, è consentito a qualsiasi codice che può farvi riferimento.</span><span class="sxs-lookup"><span data-stu-id="7265c-106">However, `derivedClass` can be accessed from any code that can reference it.</span></span> <span data-ttu-id="7265c-107">Di conseguenza, `derivedClass` non può ereditare da `baseClass` se usa `restrictedStructure` come argomento di tipo perché altrimenti `restrictedStructure` verrebbe esposta al codice in qualsiasi assembly.</span><span class="sxs-lookup"><span data-stu-id="7265c-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `restrictedStructure` as a type argument, because that could expose `restrictedStructure` to code in any assembly.</span></span>  
  
 <span data-ttu-id="7265c-108">**ID errore:** BC30922</span><span class="sxs-lookup"><span data-stu-id="7265c-108">**Error ID:** BC30922</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="7265c-109">Per correggere l'errore</span><span class="sxs-lookup"><span data-stu-id="7265c-109">To correct this error</span></span>  
  
-   <span data-ttu-id="7265c-110">Modificare i livelli di accesso delle classi o delle interfacce in modo che il tipo derivato non espanda il livello di accesso del tipo limitato.</span><span class="sxs-lookup"><span data-stu-id="7265c-110">Adjust the access levels of the classes or interfaces so the derived type does not expand the access level of the restricted type.</span></span>  
  
     <span data-ttu-id="7265c-111">-oppure-</span><span class="sxs-lookup"><span data-stu-id="7265c-111">-or-</span></span>  
  
-   <span data-ttu-id="7265c-112">Se non è possibile modificare i livelli di accesso, non usare il tipo limitato come argomento di tipo quando si costruisce la classe o l'interfaccia base.</span><span class="sxs-lookup"><span data-stu-id="7265c-112">If you cannot adjust the access levels, do not use the restricted type as a type argument when constructing the base class or interface.</span></span>
