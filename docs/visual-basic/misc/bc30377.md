# <a name="method-39ltmethodnamegt39-is-already-declared-in-interface-39ltinterfacenamegt39"></a>Metodo &#39; &lt;NomeMetodo&gt;&#39; è già dichiarato nell'interfaccia &#39;&lt; InterfaceName&gt;&#39;
Un metodo di interfaccia è stato dichiarato una seconda volta e quindi duplicato.  
  
 **ID errore:** BC30377  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere la dichiarazione duplicata.
