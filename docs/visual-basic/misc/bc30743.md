# <a name="type-39lttypenamegt39-is-not-defined-or-the-module-containing-it-is-not-loaded-in-the-debugging-session"></a>Tipo &#39; &lt;typename&gt; &#39; non è definito oppure il modulo che lo contiene non è caricato nella sessione di debug
È stato fatto riferimento a un tipo che non è disponibile o non è definito.  
  
 **ID errore:** BC30743  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Verificare che il nome del tipo sia digitato correttamente.  
  
2.  Definire il tipo all'interno della classe corrente o del modulo corrente.
