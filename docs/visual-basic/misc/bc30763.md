# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-using-the-ansi-auto-or-unicode-modifier"></a>La versione di destinazione di .NET Compact Framework non supporta l'utilizzo del modificatore ANSI, Auto o Unicode
La versione di .NET Compact Framework in uso non supporta i modificatori `Ansi`, `Auto`e `Unicode` .  
  
 **ID errore:** BC30763  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
- Rimuovere il modificatore.
