# <a name="arrays-cannot-be-declared-with-new"></a><span data-ttu-id="06723-102">Le matrici non possono essere dichiarate con 'New'</span><span class="sxs-lookup"><span data-stu-id="06723-102">Arrays cannot be declared with 'New'</span></span>
<span data-ttu-id="06723-103">La parola chiave `New` può essere presente solo nella parte relativa all'inizializzazione di una dichiarazione di matrice.</span><span class="sxs-lookup"><span data-stu-id="06723-103">The `New` keyword can appear only in the initialization part of an array declaration.</span></span> <span data-ttu-id="06723-104">Ciò significa che `New` deve essere a destra del segno di uguale (`=`) così da poter creare un nuovo tipo di matrice da assegnare alla variabile di matrice.</span><span class="sxs-lookup"><span data-stu-id="06723-104">This means `New` must be on the right side of the equal sign (`=`) so it can create a new array type to be assigned to the array variable.</span></span>  
  
 <span data-ttu-id="06723-105">Il collegamento per l'inizializzazione della classe non è disponibile per le matrici.</span><span class="sxs-lookup"><span data-stu-id="06723-105">The shortcut for class initialization is not available for arrays.</span></span> <span data-ttu-id="06723-106">Le seguenti righe di codice sono entrambe valide e sono equivalenti perché inizializzano un oggetto da una classe.</span><span class="sxs-lookup"><span data-stu-id="06723-106">The following two code lines are both valid and are equivalent because they initialize an object from a class.</span></span>  
  
```  
Dim formA as Form = New Form  
Dim formA as New Form  
```  
  
 <span data-ttu-id="06723-107">Tuttavia, l'inizializzazione di matrice non può usare lo stesso collegamento dell'inizializzazione della classe.</span><span class="sxs-lookup"><span data-stu-id="06723-107">However, array initialization cannot use the same shortcut as class initialization.</span></span>  
  
 <span data-ttu-id="06723-108">Si noti che la clausola `New` per la matrice deve contenere sia le parentesi tonde, `()`, che le parentesi graffe, `{}`.</span><span class="sxs-lookup"><span data-stu-id="06723-108">Note that the `New` clause for an array must contain both parentheses, `()`, and braces, `{}`.</span></span> <span data-ttu-id="06723-109">Le parentesi tonde specificano che il nuovo tipo è una matrice e le parentesi graffe forniscono i valori di inizializzazione.</span><span class="sxs-lookup"><span data-stu-id="06723-109">The parentheses specify that the new type is an array, and the braces supply the initialization values.</span></span> <span data-ttu-id="06723-110">Il compilatore richiede le parentesi graffe anche se sono vuote, vale a dire, anche se non viene inizializzato nessuno dei valori di matrice.</span><span class="sxs-lookup"><span data-stu-id="06723-110">The compiler requires the braces even if they are empty, that is, even if you are not initializing any of the array values.</span></span>  
  
 <span data-ttu-id="06723-111">**ID errore:** BC30053</span><span class="sxs-lookup"><span data-stu-id="06723-111">**Error ID:** BC30053</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="06723-112">Per correggere l'errore</span><span class="sxs-lookup"><span data-stu-id="06723-112">To correct this error</span></span>  
  
-   <span data-ttu-id="06723-113">Sostituire un'istruzione come `Dim myDates() As New Date` con un'istruzione come `Dim myDates() As Date = New Date() {}`.</span><span class="sxs-lookup"><span data-stu-id="06723-113">Replace a statement such as `Dim myDates() As New Date` with a statement such as `Dim myDates() As Date = New Date() {}`.</span></span>
