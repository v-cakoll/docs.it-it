# <a name="39redim39-statements-can-no-longer-be-used-to-declare-array-variables"></a>&#39;ReDim&#39; non è più possibile utilizzare le istruzioni per dichiarare le variabili di matrice
`ReDim` può essere usato solo per modificare le dimensioni di una matrice esistente.  
  
 **ID errore:** BC30811  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Specificare le dimensioni delle matrici quando vengono dichiarate, ad esempio:  
  
    ```  
    Dim X(20) As Integer  
    ```
