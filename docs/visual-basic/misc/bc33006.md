# <a name="39end-operator39-must-be-the-first-statement-on-a-line"></a>&#39;Terminare l'operatore&#39; deve essere la prima istruzione in una riga
Un'istruzione `End Operator` segue un separatore di istruzioni due punti (`:`). `End Operator` deve essere l'unica istruzione nella riga di origine.  
  
 **ID errore:** BC33006
