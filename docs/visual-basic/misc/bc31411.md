# <a name="typename-must-be-declared-mustinherit-because-it-contains-methods-declared-mustoverride"></a>'\<nomeTipo >' deve essere dichiarato 'come MustInherit' perché contiene metodi dichiarati come 'MustOverride'
I tipi che contengono i membri `MustOverride` devono essere dichiarati come `MustInherit`.  
  
 **ID errore:** BC31411  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Aggiungere il modificatore `MustInherit` al tipo.
