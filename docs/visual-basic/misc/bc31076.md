# <a name="ltmodulenamegt-is-a-module-and-cannot-be-referenced-as-an-assembly"></a>«&lt;modulename&gt;' è un modulo e non è possibile farvi riferimento come assembly
Si è provato a usare un modulo come assembly.  
  
 **ID errore:** BC31076  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Modificare il riferimento a un assembly.
