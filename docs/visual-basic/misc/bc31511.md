# <a name="attribute-member-ltmembernamegt-cannot-be-the-target-of-an-assignment-because-it-is-not-declared-public"></a>Membro dell'attributo '&lt;nomemembro&gt;' non può essere la destinazione di un'assegnazione perché non è dichiarato 'Public'
Si è provato ad assegnare un valore a un membro privato di un attributo.  
  
 **ID errore:** BC31511  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
1.  Rimuovere l'assegnazione.  
  
2.  Se si usano gli attributi personalizzati che sono stati sviluppati, modificare il modificatore di accesso del membro dell'attributo in `Public`.
