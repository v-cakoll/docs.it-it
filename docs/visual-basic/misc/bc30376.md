# <a name="39new39-cannot-be-used-on-class-39ltclassnamegt39-because-it-contains-a-39mustoverride39-member-that-has-not-been-overridden"></a>&#39;Nuova&#39; non può essere utilizzato nella classe &#39; &lt;classname&gt; &#39; perché contiene un &#39;MustOverride&#39; membro che non è stata sottoposta a override
Si è provato a usare `New` in una classe che contiene un membro `MustOverride` non sottoposto a override.  
  
 **ID errore:** BC30376  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Rimuovere l'istruzione `New` .
