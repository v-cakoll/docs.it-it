# <a name="unable-to-find-required-file-39ltfilenamegt39"></a>Impossibile trovare il file &#39; &lt;filename&gt;&#39;

 **ID errore:** BC30655  
  
## <a name="to-correct-this-error"></a>Per correggere l'errore  
  
-   Reinstallare Visual Studio.
